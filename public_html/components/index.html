Aim

This sites aim is to showcase HTML 5 accessibility features in web browsers. So that everyone equally is able to browse the site despite any barriers that users may have to overcome.
  • Page Aim's:

    1. Home
      • Briefly describes what each page's purpose is.
    2. About
      • Contains background information regarding web accessibility.
    3. Media
      • Demonstates HTML 5 video and audio.
    4. Gallery
      • Demonstates HTML 5 Images.
    5. Forms
      • Demonstates HTML 5 Forms.
    6. Resources
      • Contains links to accessibility related tools and articles.
Try browsing with a screen reader extension to witness the site in action, and don't forget to check out the semantic markup with your browsers developer tools web inspector .
